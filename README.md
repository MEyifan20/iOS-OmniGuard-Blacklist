## 🛠️ 配置建议 (以 AdGuard Pro 为例)
1. **添加路径**：防护 (盾牌图标) -> DNS 防护 -> DNS 过滤 -> DNS 过滤器 -> 添加过滤器。
2. **配合黑名单**：建议配合激进全量如（217heidai）型黑名单使用，本规则将自动对关键服务进行“补丁式”放行。
3. **手动刷新**：若遇到 CDN 缓存延迟，可尝试切换原生地址进行即时同步。
